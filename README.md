# S.C.A.L.E. - Smart Coin Allocation & Level Evaluator
S.C.A.L.E. (Smart Coin Allocation &amp; Level Evaluator) è un salvadanaio intelligente IoT sviluppato per il corso di Digital Programmable Systems, progettato per automatizzare la classificazione, il conteggio e la messa in sicurezza del contante metallico.
